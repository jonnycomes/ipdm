Some sets of numbers are denoted with special notation. For example, we write \(\mathbb{R}\) for the set of all real numbers 1 . We write \(\mathbb{N}\) for the set of all natural numbers. In other words, in this class we define
\begin{equation*} \mathbb{N}=\{1,2,3,4,5,\ldots\}. \end{equation*}
Thus \(2\in\mathbb{N}\) and \(1423\in\mathbb{N}\text{,}\) but \(-3\not\in\mathbb{N}\text{,}\) \(0.3\not\in\mathbb{N}\) and \(0\not\in\mathbb{N}\)  2 .
