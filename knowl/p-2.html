We take a set to be a primitive notion. In other words, we will not attempt to give a formal definition of a set 1 . We will simply say that a set is an unorderded collection of distinct elements. Given a set \(S\text{,}\) we write \(x\in S\) to mean that \(x\) is an element of \(S\). We write \(x\not\in S\) to means that \(x\) is not an element of \(S\).
