A rational number is a real number that can be written in the form \(\frac{a}{b}\) with \(a\in\mathbb{Z}\) and \(b\in\mathbb{N}\text{.}\) We write \(\mathbb{Q}\) for the set of all rational numbers. For example, \(\frac{1}{2}\in\mathbb{Q}\) and \(\frac{5}{3}\in\mathbb{Q}\text{.}\) We also have \(4\in\mathbb{Q}\) and \(-14\in\mathbb{Q}\) since \(4=\frac{4}{1}\) and \(-14=\frac{-14}{1}\text{.}\) It turns out that there are irrational numbers too: \(\pi\not\in\mathbb{Q}\text{,}\) \(e\not\in\mathbb{Q}\text{,}\) and \(\sqrt{2}\not\in\mathbb{Q}\text{.}\) However, proving a number is irrational requires proof (see Example ???).
